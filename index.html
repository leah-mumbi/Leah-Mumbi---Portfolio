🛠️ Projects

Project 1: Grocery List Manager 🛒

Grocery List Manager is a simple and efficient web application that helps users create, manage, and organize their grocery shopping lists. Users can add items, remove items, and submit the final list for easy reference while shopping. The project enhances user experience by providing an intuitive interface, making grocery planning hassle-free.

Key Features:

  • ✔ Add Items – Easily input grocery items to build your list.
  • ✔ Delete Items – Remove unwanted items with a simple click.
  • ✔ Submit List – Finalize and save your grocery list for shopping.
  • ✔ User-Friendly Interface – Clean, minimalistic design for smooth navigation.

Technologies Used:

HTML, CSS, JavaScript (Frontend) | Local Storage (for saving the list temporarily)

Project 2: ChocoMelt Adventure! 🎮🍫

Players navigate through exciting levels, unlock rewards, and enjoy an immersive digital adventure.

HTML, CSS, JavaScript | Game Logic (JavaScript) | Animations (CSS & JS) | Audio Integration
